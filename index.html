Welcome to the UntitledDesktopEnvironment RPM repository

This repository is used for hosting all UntitledDesktopEnvironment packages, as well as all MadLadSquad applications.

To use this repository use the following configuration to /etc/yum.repos.d/ude-rpms.repo :

[ude-rpms]
name=UntitledDesktopRPM
baseurl=https://rpms.madladsquad.com/
enabled=1
gpgcheck=0

Available packages:

  • untitled-game-system-manager - Provides UntitledGameSystemManager, a manager for containerised Linux game systems
  • untitled-ibus-handwriting - Provides UntitledIBusHandwriting, an IBus handwriting input method for IBus
  • ude-session-logout - Provides UntitledDESessionLogout, a session logout utility for the UntitledDesktopEnvironment
  • untitled-imgui-framework - Provides UntitledImGuiFramework, an independent cross-platform desktop application development toolkit
  • untitled-dbus-utils - Provides UntitledDBusUtils, a C++ wrapper on top of the low level D-Bus C API
  • untitled-exec - Provides UntitledExec, a cross-platform fork and exec/CreateProcess wrapper library
  • untitled-cli-parser - Provides UntitledCLIParser, a parser for CLI arguments
  • untitled-i18n - Provides UntitledI18N, a i18n library
  • untitled-open - Provides UntitledOpen, a file dialog, URL and file opener library
  • untitled-xdg-basedir - Provides UntitledXDGBasedir, a better implementation of the XDG base directory specification
